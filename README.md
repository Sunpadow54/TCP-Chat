# A Simple Chat

## *This is a simple command-line chat IRC like, using TCP Net socket, and connection (unique username & a password)*

-----------------


## :wrench: Installation

Use **node version 14.0** 

```
npm install
```


### :arrow_forward: How to Run locally

```
npm run server
> to run the server

---------------------

npm run client || node client.js
> to run a client
```

Type commands:
```
/quit
> to leave the chat
```

## :package: Made with

* Node v14.0.0 
    * Net Module
